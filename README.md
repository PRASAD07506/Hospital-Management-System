# 🏥 Hospital Management System

A complete **Hospital Management System (HMS)** developed using PHP and MySQL to manage patients, doctors, and hospital operations efficiently.

---

## 🚀 Features

### 👤 Patient Module

* Patient Registration & Login
* Book Appointments
* View Medical History
* Access Dashboard

### 👨‍⚕️ Doctor Module

* Manage Patient Records
* View Appointments
* Update Treatment Details

### 🛠️ Admin Module

* Manage Doctors & Patients
* Handle Appointments
* System Dashboard
* Data Management

## 🛠️ Installation Guide

1. Install XAMPP

2. Move project to:

   ```
   C:\xampp\htdocs\
   ```

3. Start:

   * Apache
   * MySQL

4. Open phpMyAdmin:

   * Create a database
   * Import SQL file (if available)

5. Update database config:

   ```
   config/db.php
   ```

6. Run project:

   ```
   http://localhost/hms
   ```
